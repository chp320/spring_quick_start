컬렉션(Collection) 객체 설정
- 배열이나 List 같은 컬렉션(collection) 객체를 이용하여 데이터 결합을 사용해야 하는 경우 존재
- 이 경우, 컬렉션 객체를 의존성 주입하면 되는데, 이를 위해 컬렉션 맵핑과 관련된 엘리먼트 확인

1. List 타입 맵핑
- 배열 객체나 java.util.List 타입의 컬렉션 객체

2. Set 타입 맵핑
- 중복 값을 허용하지 않는 집합 객체 사용 시 java.util.Set 컬렉션 사용
- <set> 태그 사용

3. Map 타입 맵핑
- 특정 key로 데이터를 등록/사용 시 java.util.Map 컬렉션 사용
- <map> 태그 사용
