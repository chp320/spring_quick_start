웹 소켓(Web Socket) API

  • 서버와 클라이언트 간에 실시간으로 데이터를 교활할 수 있게 해주는 기술
  • HTTP 프로토콜인 Request와 Response 기반으로 데이터 송수신
  • 기본적인 흐름 (클라이언트 - 서버)
    • (client) TCP/IP 접속 요청
    • (server) TCP/IP 접속 수락
    • (client) 웹 소켓 열기 (핸드쉐이크 요청)
    • (server) 웹 소켓 열기 (핸드쉐이크 수락)
    • (client/server) 데이터 송.수신
  • 웹 소켓 형식
    • [형식]
    var 객체명 = new WebSocket("ws://사용자 도메인/페이지") ;
    ➞ 일반 통신은 "ws://", 보안(SSL)은 "wss://" 프로토콜을 사용!
  • 웹 소켓 객체
    • 이벤트 또는 메서드 설명
    onopen 서버와 연결이 설정되면 발생하는 이벤트
    onclose 서버와 연결이 끊어지면 발생하는 이벤트
    onmessage 서버에서 push(전송)하는 데이터를 받는 이벤트
    onerror 에러가 발생할 때 받는 이벤트
    send() 데이터를 송신하는 메서드
    close() 객체 종료 메서드
