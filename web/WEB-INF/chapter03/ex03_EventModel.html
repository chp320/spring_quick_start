이벤트(Event) 모델

  • 이벤트: 키보드 입력, 마우스 클리과 같이 사용자가 특정 행위를 했을 때 행위에 대한 결과 동작을 보여주는 것
  • 이벤트 모델은 이벤트 이름/이벤트 속성/이벤트 핸들러.로 구성된다.

JavaScript에서의 이벤트 유형

  • 마우스 이벤트
  • 키보드 이벤트
  • 포커스 이벤트
  • 폼 이벤트
  • 유저 인터페이스(UI, User Interface) 이벤트
  • 드래그 이벤트
  • 터치 이벤트
