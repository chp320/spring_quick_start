표준객체

  • JavaScript에서 자주 사용하는 Number, String, Array, Date, Math 등 내장 객체를 의미

Array 객체

  • 같은 타입의 자료만 저장 가능한 다른 언어와 달리, 서로 다른 타입의 데이터 저장 가능하다.
  • 리터럴형 : 일반적인 방식. 대괄호([])를 사용하여 배열 선언과 값을 한 번에 처리
  • 배열객체형: new 키워드를 사용하여 객체 선언과 동시에 값을 인자로 추가

형식

(리터럴형) var 변수명 = [값1, 값2, 값3, ... 값n] ;
(배열객체형) var 변수명 = new Array(값1, 값2, 값3, ... 값n) ;

Array 관련 메서드

  • sort() - 배열 값들을 오름차순으로 정렬
  • reverse() - 배열 값들을 역순으로 변경
  • concat(array) - 2개의 배열을 합하여 하나의 배열로 만듦
  • join([str]) - 배열에 들어 있는 값을 모두 붙혀서 하나의 문자열로 만듦
  • push() - 배열의 마지막에 새로운 원소(값)을 추가
  • pop() - 배열의 마지막 원소(값)을 추출
