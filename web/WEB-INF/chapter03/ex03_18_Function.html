함수(Function)

함수는 선언적 함수와 익명 함수로 작성할 수 있다.

선언적 함수

  • 함수 선언식(Function Declarations)라고도 한다.
  • function 이라는 키워드를 사용해서 함수를 선언한다.
  • 호출문이 먼저 나와도 에러 없이 처리 가능하며, 일반적으로 많이 사용하는 방식이다.

함수의 정의 및 호출

  • 정의: function 함수명 { 실행문; }
  • 호출: 함수명() ;

매개변수와 리턴값

  • 매개변수(인자) : 호출하는 함수에 값을 전달할 때 넣어 보내는 변수
  • 리턴값 : 함수에서 반환하는 값
  • 리턴값이 없는 경우도 있으며, return 키워드만 쓰거나 생략 가능하다.

익명 함수

  • 함수 표현식(Function Expressions)라고도 한다.
  • function 키워드를 사용하지만 함수명이 없는 형태이다.

형식

var 변수명 = function() { 실행문 ; }

내장 함수

  • JavaScript가 자체적으로 가지고 있는 함수
  • 네트워크 통신용 함수와 숫자와 관련된 함수 존재

    • 네트워크 통신용 함수

    • encodeURI() - 파라미터를 전달하는 URI 전체를 인코딩 시 사용. 특수문자를 제외한 '문자'만 인코딩
    • decodeURI() - encodeURI()로 인코딩된 데이터를 다시 되돌림

    숫자와 관련된 함수

    • isNan() - NaN은 'Not a Number'의 약자. 숫자가 아닌 문자가 포함되면 true를 반환.
    • isFinite() - 값이 유한수인지를 판단.
    • parseInt() - 문자를 숫자로 변환.
    • parseFloat() - 문자를 실수형으로 변환.
    • eval() - 문자로 된 수식을 JavaScript의 수식으로 인식하여 실행 및 결과를 반환.
    • Number() - 문자를 숫자형으로 변환.
    • String() - 숫자를 문자형으로 변환.
