문서 객체 모델(DOM)

근거리 선택자

  • ownerDocument - 최상위 노드를 선택
  • parentNode - 선택요소의 부모 노드를 선택
  • childNodes - 선택요소의 "모든" 자식 노드를 선택 (배열 객체로 저장)
  • children - 선택요소의 자식 노드만 선택 (배열 객체로 저장)
  • firstChild - 선택요소의 첫 번째 자식 노드만 선택
  • lastChild - 선택요소의 마지막 자식 노드만 선택
  • previousSibling - 선택요소의 이전에 오는 형제 노드를 선택
  • nextSibling - 선택요소의 다음에 오는 형제 노드를 선택

DOM 객체 메서드

  • createElement(tagName) - 요소 노드를 생성
  • createTextNode(text) - 텍스트 노드를 생성
  • removeChild(child) - 자식 노드를 삭제
  • appendChild(node) - 자식 노드를 추가
  • setAttribute(name, value) - 객체의 속성을 설정
  • getAttribute(name) - 객체의 속성을 가져옴
