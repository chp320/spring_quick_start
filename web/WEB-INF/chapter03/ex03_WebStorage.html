웹 스토리지

  • 정의:
    • 브라우저(또는 클라이언트) 내부에 스토리지라는 저장소를 제공해주는 기능
    • 서버가 아닌 클라이언트에 데이터를 저장
  • 쿠키 vs 웹 스토리지
    • 쿠키: 사용기한이 있는 저장소. 매번 서버 요청 시 같이 전송해야 하기 때문에 웹이 느려지는 원인. 서버에 http 요청 시 암호화되지 않기 때문에 보안에 매우 취약
    • 웹 스토리지: 서버로 '절대' 보내지 않기 때문에 사용자가 많은 양의 정보를 안전하게 보관 가능
    • 웹 스토리지는 key와 value로 데이터를 저장하며, key를 통해 value를 꺼낼 수 있다.
    • 웹 스토리지는 세션 스토리지(session storage)와 로컬 스토리지(local storage)가 있고, 모두 클라이언트에 저장됨
