표준객체

  • JavaScript에서 자주 사용하는 Number, String, Array, Date, Math 등 내장 객체를 의미

Date 객체

  • 날짜와 시간에 대한 정보 제공
  • 객체 생성 시 날짜 미지정 : 시스템에 설정된 날짜/시간으로 설정

형식

(시스템 정보 제공) var 변수명 = new Data() ;
(설정 값 제공) var 변수명 = new Data(년, 월, 일, 시, 분, 초) ;

날짜/시간 관련 메서드

  • setFullYear() / getFullYear() - '년도'만 설정/반환
  • setMonth() / getMonth() - '월'만 설정/반환
  • setDate() / getDate() - '일(월 기준)'만 설정/반환
  • setDay() / getDay() - '요일'만 설정/반환 (단, 일~토가 인덱스로 반환)
  • setHour() / getHour() - '시간'만 설정/반환
  • setMinutes() / getMinutes() - '분'만 설정/반환
  • setSeconds() / getSeconds() - '초'만 설정/반환
  • setMiliseconds() / getMiliseconds() - '밀리 초'만 설정/반환
  • toString() - 날짜를 문자형식으로 반환
