브라우저 객체 모델(BOM)

정의

  • 웹 브라우저와 관련된 객체들의 집합
  • JavaScript에는 브라우저의 정보에 접근하거나, 기능적 요소들을 직접 제어하는 등 관리할 수 있는 기능이 존재
  • location 객체 : URL의 정보를 제공
  • navigator 객체 : 현재 실행 중인 브라우저의 정보를 제공
  • history 객체 : 방문 기록에 대한 정보를 제공
  • screen 객체 : 모니터의 정보를 제공
  • document 객체 : 문서에 대한 정보를 제공
  • BOM은 전역 객체(Global Object)인 window 객체 아래에 존재한다.

window 객체

  • 브라우저 객체 모델(BOM)의 최상위 객체
  • window의 형태와 위치를 변경할 수 있는 메서드

location 객체

  • URL과 관련된 인터넷의 주소와 관련된 속성
  • 프로토콜, 호스트 이름, 문서 주소 등의 정보가 존재
