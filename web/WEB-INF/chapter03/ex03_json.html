JSON(Javascript Object Notation)

  • JSON은 클라이언트와 서버의 통신에서 데이터를 주고 받을 때 주로 사용
  • 특징
    • JSON은 javascript를 기반으로 만들어졌으며, 특정 언어에 종속되지 않음
    • key, value의 쌍으로 구성되고, key와 value는 콜론(:)으로 구분
    • key는 문자열로 구성되기 때문에 쌍땨옴표(" ") 또는 단일 따옴표(' ')를 사용
  • 작성 방법
    • 1. 객체인 경우
    {"이름1":"값1", "이름2":"값2", ..., "이름n":값n"}

    2. 배열인 경우
    [{"이름1":"값1", "이름2":"값2"}, {"이름1":"값1", ..., "이름n":"값n"}]

    [사용방법]
    객체명.이름 또는 객체명['이름']

    [예시]
    1. 객체인 경우
    var book = {"code":101, "title":java, "page":350}

    2. 배열인 경우
    var book = [{"code": 101, "title":java, "page":350}, {"code":102, "title":jsp, "page":500}]
