정규 표현식(Regular Expression)

  • 특정한 규칙을 가진 문자열의 집합
  • 조합된 문자열이 특정 규칙에 맞게 작성되었는지를 찾아내는 검색 패턴
  • 정규 표현식 만드는 2가지 방법
    • RegExp 객체를 이용한 생성
    • 정규 표현식 리터럴을 이용한 생성
  • 형식
    • var 객체명 = new RegExp('정규표현식', '[Flag']) ;
    var 객체명 = /정규표현식/[Flag] ;
  • Flag 종류
    • Flag 설명
    g 문자열 내의 모든 패턴을 찾아 줍니다.
    i 문자열의 대,소문자를 구별하지 않습니다.
    m 문자열의 줄바꿈이 일어나도 찾아줍니다.
  • 정규 표현식의 패턴
    • 표현식 설명 표현식 설명
    ^ 문자열의 시작을 의미합니다. | or를 의미합니다.
    $ 문자열의 종료를 의미합니다. \d 숫자[0-9]와 동일합니다.
    . 임의의 한 문자입니다. \s 공백 문자입니다.
    * 앞 문자가 없을 수도 있고 많을 수도 있습니다. \S 공백 문자가 아님을 의미합니다.
    + 반복을 표현하며, 앞 문자가 한 번 이상 반복함을 의미합니다. \w 단어를 만들 수 있는 알파벳 또는 숫자입니다.
    ? 존재 여부를 표히사며 앞 문자가 없거나 있습니다. \W 알파벳, 숫자를 제외한 문자 입니다.
    {}} 횟수 또는 범위를 나타냅니다. \D 숫자를 제외한 모든 문자입니다.
    ()) 그룹을 의미합니다. \ \다음에 특수문자가 오면 그 문자 자체를 의미합니다.
    []] 문자의 집합이나 범위를 나타내며 두 문자 사이의 - 기호로 범위를 나타냅니다.
  • 간단한 사용 예제
    • ex) ^[0-9]*$
    ^ : 문자열의 시작을 알림
    [0-9] : 숫자 범위 지정
    * : 문자수는 상관 없음
    $ : 문자열의 종료를 알림
