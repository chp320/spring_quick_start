문서 객체 모델(DOM, Document Object Model)

  • document와 관련된 집합
  • DOM을 사용해서 HTML 페이지에 태그를 추가, 삭제, 수정 가능
  • HTML의 DOM 구조에서 최상위는 #document이고 마지막은 #text로 구성됨
  • document 객체의 요소를 추출하는 메서드 선택자 : 원거리 선택자, 근거리 선택자
    • 원거리 선택자: 요소의 '속성명'으로 선택
    • 근거리 선택자: 가까이에 있는 요소를 선택

원거리 선택자

  • getElementById("id명") - 태그의 id명이 일치하는 문서 객체를 가져옴
  • getElementsByName("name명") - 태그의 name명이 일치하는 문서 객체를 가져옴
  • getElementsByTagName("tag명") - 태그의 tag명이 일치하는 문서 객체를 가져옴
